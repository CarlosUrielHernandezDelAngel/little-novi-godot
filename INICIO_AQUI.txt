🔧 FUNCIONES PÚBLICAS CLAVE:

  guardar_progreso_en_slot(numero_slot: int) → bool
  └─ Guarda datos + captura pantalla + miniatura

  cargar_progreso_de_slot(numero_slot: int) → Dictionary
  └─ Carga datos del slot

  actualizar_progreso_actual(datos: Dictionary) → void
  └─ Actualiza progreso en memoria

  obtener_informacion_todos_slots() → Array
  └─ Información de todos los slots

  eliminar_slot(numero_slot: int) → bool
  └─ Borra completamente un slot

  obtener_slot_seleccionado() → int
  └─ Retorna número del slot actual

❓ PREGUNTAS FRECUENTES:

  P: ¿Cómo cambio el tamaño de las miniaturas?
  R: En slots.gd, modifica: const TAMAÑO_MINIATURA = Vector2(200, 150)

  P: ¿Puedo tener más de 3 slots?
  R: Sí, cambia: const NUMERO_SLOTS = 5 (y crea más contenedores en la escena)

  P: ¿Dónde se guardan los archivos?
  R: En user:// (carpeta local del proyecto, típicamente AppData en Windows)

  P: ¿Se guardan automáticamente?
  R: No, debes llamar a guardar_progreso_en_slot() manualmente. Ve el ejemplo.

  P: ¿Cómo acceso los datos desde otro script?
  R: var slots = get_tree().root.get_node("Control")
     var progreso = slots.obtener_progreso_actual()
