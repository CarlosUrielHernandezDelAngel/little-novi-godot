# Little Novi Game

**Little Novi Game** es una novela visual 2D desarrollada en **Godot 4**, creada como proyecto académico.  
El juego incluye narrativa interactiva, sistema de login, menús y guardado/carga de partidas.

---

## Requisitos del sistema
- **Sistema operativo**: Windows 10/11, Linux o macOS.
- **Motor de juego**: [Godot Engine 4.x](https://godotengine.org/download).
- **Espacio en disco**: mínimo 200 MB libres.
- **Memoria RAM**: 4 GB recomendados.

---

## Instalación (Método más sencillo: Descargar ZIP

1. Haz clic en el botón verde **Code** y selecciona **Download ZIP**.

2. Extrae el archivo ZIP en tu computadora (ejemplo: en `Documentos/LittleNoviGame`).

3. Abre **Godot Engine V. 4.6.3** (previamente descargado desde el link adjunto).

4. En la pantalla inicial de Godot, selecciona **Importar proyecto**.

5. Busca la carpeta extraída y selecciona el archivo `project.godot`.

6. Haz clic en **Play (un icono de triángulo)** para ejecutar el juego.

---

## Uso del juego
- **Login**: al iniciar, crea un usuario para guardar tu progreso.  
- **Menú principal**: opciones de *Inicio*, *Cargar partida* y *Acerca de*.  
- **Sistema de guardado/carga**: usa los slots disponibles para almacenar tu progreso.  
- **Menú flotante**: accesible durante el juego para guardar, cargar o regresar al menú.  

---

## Solución de problemas comunes
- **El proyecto no abre** → asegúrate de usar Godot 4.6.3
- **No se ven los assets** → revisa que hayas extraído el ZIP completo.  
- **Error al guardar/cargar** → verifica permisos de escritura en tu carpeta de usuario.
